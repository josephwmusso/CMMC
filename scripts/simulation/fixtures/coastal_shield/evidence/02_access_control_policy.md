Coastal Shield's Access Control Policy defines role-based access via
Microsoft Entra ID security groups. Quarterly access reviews are
conducted by the IT Manager using Entra ID Access Reviews. JML
(Joiner-Mover-Leaver) procedures require same-day deprovisioning
via automated Entra ID lifecycle workflows. Privileged accounts use
FIDO2 security keys and are limited to 3 named administrators.
